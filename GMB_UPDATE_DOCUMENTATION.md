## How to Update GMB Website Field

Follow these steps to update the website URL for each GMB location:

### Step 1: Access Google My Business

1. Go to [Google Business Profile Manager](https://business.google.com/)
2. Sign in with the Google account that manages Scrapiz locations
3. If you manage multiple locations, you'll see a list of all businesses

### Step 2: Select a Location

1. Click on the location you want to update (e.g., "Scrapiz Bandra")
2. This will open the business profile dashboard for that location

### Step 3: Edit Business Information

1. In the left sidebar, click on **"Info"** or **"Business Information"**
2. Scroll down to find the **"Website"** field
3. Click the pencil/edit icon next to the Website field

### Step 4: Update the Website URL

1. Clear any existing URL (if it's pointing to the homepage or incorrect page)
2. Enter the correct location-specific URL from the table above
3. Ensure the URL is exactly as shown (including `https://www.scrapiz.in/`)
4. Click **"Apply"** or **"Save"** to confirm the change

### Step 5: Verify the Update

1. After saving, refresh the page
2. Verify the website URL is correctly displayed
3. Click the website link to ensure it opens the correct location page
4. Mark the location as "✅ Updated" in the status checklist above

### Step 6: Repeat for All Locations

Repeat Steps 2-5 for all 9 locations until all are updated.

## Additional Notes

### Important Reminders

1. **Do Not Use Homepage**: Never point GMB listings to the homepage (`https://www.scrapiz.in/`). Always use the location-specific page.

2. **NAP Consistency**: Ensure the business name, address, and phone number on the GMB listing exactly match what's displayed on the location page.

3. **Schema Markup**: Each location page includes LocalBusiness schema markup with the same NAP information as GMB.

4. **Mobile Testing**: After updating, test each GMB listing on mobile devices to ensure the website link works correctly.

5. **Monitor Performance**: After updates, monitor Google Search Console and GMB Insights to track improvements in local search performance.

### Troubleshooting

**Issue:** Website field is grayed out or can't be edited
- **Solution:** Ensure you have the correct permissions (Owner or Manager role) for the GMB listing

**Issue:** Changes don't appear immediately
- **Solution:** GMB updates can take 24-48 hours to propagate. Check back later.

**Issue:** URL shows as invalid
- **Solution:** Ensure the URL includes `https://` and has no typos. Copy-paste from this document.

**Issue:** Location page shows 404 error
- **Solution:** Verify the location page is deployed and accessible. Check the website's routing configuration.
