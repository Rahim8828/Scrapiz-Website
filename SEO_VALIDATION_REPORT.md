## Validation Instructions

### Step 1: Start Development Server
```bash
npm run dev
```

### Step 2: For Each Location Page

1. **Open the location page** in your browser (use local URL)
2. **View page source** (Right-click ‚Üí View Page Source or Ctrl+U)
3. **Copy the schema markup** from the `<script type="application/ld+json">` tag
4. **Go to Google Rich Results Test**: https://search.google.com/test/rich-results
5. **Select "CODE" tab** and paste the schema markup
6. **Click "TEST CODE"** button
7. **Review results:**
   - Check for errors (red indicators)
   - Check for warnings (yellow indicators)
   - Verify all expected schema types are recognized
8. **Document findings** in this report
9. **Take screenshots** if issues are found

### Step 3: Alternative - Test Live URL
If the site is already deployed:
1. Go to Google Rich Results Test
2. Select "URL" tab
3. Enter the full URL (e.g., https://www.scrapiz.in/bandra)
4. Click "TEST URL"
5. Review and document results

### Step 4: Schema.org Validation (Optional)
For additional validation:
1. Go to https://validator.schema.org/
2. Paste the schema markup
3. Review any additional warnings or suggestions

## Expected Schema Structure

Each location page should have a schema with `@graph` containing:

### 1. RecyclingCenter (LocalBusiness)
```json
{
  "@type": "RecyclingCenter",
  "@id": "https://www.scrapiz.in/{location}#business",
  "name": "Scrapiz {Location} - Scrap Buyers & Scrap Dealers in Mumbai",
  "description": "...",
  "image": "...",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "...",
    "addressLocality": "Mumbai",
    "addressRegion": "Maharashtra",
    "postalCode": "...",
    "addressCountry": "IN"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": ...,
    "longitude": ...
  },
  "telephone": "+91...",
  "email": "Contact@scrapiz.in",
  "url": "https://www.scrapiz.in/{location}",
  "openingHoursSpecification": [...],
  "priceRange": "$",
  "areaServed": {
    "@type": "City",
    "name": "{Location}"
  }
}
```

### 2. Service
```json
{
  "@type": "Service",
  "@id": "https://www.scrapiz.in/{location}#service",
  "serviceType": "Scrap Collection and Recycling",
  "provider": {
    "@id": "https://www.scrapiz.in/{location}#business"
  },
  "areaServed": {
    "@type": "City",
    "name": "{Location}"
  },
  "hasOfferCatalog": {
    "@type": "OfferCatalog",
    "name": "Scrap Materials",
    "itemListElement": [...]
  }
}
```

### 3. FAQPage
```json
{
  "@type": "FAQPage",
  "@id": "https://www.scrapiz.in/{location}#faq",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "...",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "..."
      }
    }
  ]
}
```
